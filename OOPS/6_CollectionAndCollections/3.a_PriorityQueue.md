## PriorityQueue:
1. Elements are processed based on priority rather than insertion order.
2. By default, it uses natural ordering (min-heap), but a custom comparator can be used to define different priorities.
3. Supports standard queue operations like add(), poll(), and peek().
4. Automatically grows as elements are added and null insertion is not possible .
5. Uses a heap data structure internally to ensure efficient insertion and removal of the highest-priority element.

## Internal Implementation of PriorityQeue:
PriorityQueue is internally implemented using a resizable array and organized using a heap data structure (binary heap). By default, its initial capacity is 11. When the array becomes full and a new element is inserted (for example, the 12th element), a new larger array is created, and all existing elements are copied into the new array while maintaining the heap structure.

## Constructors of PriorityQueue

**1. PriorityQueue()**
This method creates a PriorityQueue with the default initial capacity (11) that orders its elements according to their natural ordering.
```java
PriorityQueue<E> pq = new PriorityQueue<E>();
```

**2. PriorityQueue(int initialCapacity)**
This creates a PriorityQueue with the specified initial capacity that orders its elements according to their natural ordering.
```java
PriorityQueue<E> pq = new PriorityQueue<E>(initialCapacity);
```

**3. PriorityQueue(Comparator<E> comparator)**
This creates a PriorityQueue with the default initial capacity and whose elements are ordered according to the specified comparator.
```java
Comparator<Integer> c = (a, b) -> b - a; // max-heap
PriorityQueue<Integer> pq = new PriorityQueue<>(c);
```

**4. PriorityQueue(int initialCapacity, Comparator<E> comparator)**
This creates a PriorityQueue with the specified initial capacity that orders its elements according to the specified comparator.
```java
PriorityQueue<E> pq = new PriorityQueue<E>(int initialCapacity, Comparator<E> comparator);
```

## Operations on PriorityQueue

```java
import java.util.Iterator;
import java.util.PriorityQueue;

public class Main {

    public static void main(String[] args) {

        // create PriorityQueue
        PriorityQueue<Integer> pq = new PriorityQueue<>();

        // add elements
        pq.add(30);
        pq.add(10);
        pq.add(20);
        pq.add(40);

        System.out.println("PriorityQueue: " + pq);

        // peek
        System.out.println("Peek element: " + pq.peek());

        // poll
        System.out.println("Poll element: " + pq.poll());

        System.out.println("After poll: " + pq);

        // remove specific element
        pq.remove(30);

        System.out.println("After removing 30: " + pq);

        // iteration using iterator
        System.out.println("Iterating elements:");

        Iterator<Integer> it = pq.iterator();

        while(it.hasNext()) {
            System.out.println(it.next());
        }
    }
}
// Output:
// PriorityQueue: [10, 30, 20, 40]

// Peek element: 10

// Poll element: 10

// After poll: [20, 30, 40]

// After removing 30: [20, 40]

// Iterating elements:
// 20
// 40
```
