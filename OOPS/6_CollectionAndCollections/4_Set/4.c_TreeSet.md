## TreeSet:
1. It stores unique elements in a sorted order and internally uses a Red-Black tree to maintain sorting.
2. Does not allow null elements because it uses comparison for sorting, which causes NullPointerException.
3. Implements the NavigableSet interface and provides navigation methods like higher(), lower(), ceiling() and floor().
4. It is not synchronized. it must be synchronized using Collections.synchronizedSet().

## Internal Implementation of TreeSet:
1. TreeSet internally uses a TreeMap.
2. It stores elements in a self-balancing binary search tree called a Red-Black Tree.
3. Elements are ordered using Comparable or a custom Comparator.
4. It does not use hashing, buckets, hashcode, capacity, or resizing like HashSet.
5. Operations like add(), remove(), and contains() take O(log n) time.

## Constructors of TreeSet:

**1. TreeSet()**
Creates an empty TreeSet that sorts elements in their natural order
```java
TreeSet<> ts = new TreeSet<>();
```

**2. TreeSet(Comparator)**
This constructor is used to build an empty TreeSet object in which elements will need an external specification of the sorting order.
```java
TreeSet <> ts = new TreeSet<>(comp);
```

**3. TreeSet(Collection)**
Creates a TreeSet containing all elements from the given collection, stored in their natural sorted order. It is useful for converting a collection to a TreeSet.
```java
TreeSet <> t = new TreeSet<>(col);
```

**4. TreeSet(SortedSet)**
Creates a TreeSet containing the same elements and order as the specified SortedSet.
```java
TreeSet <> t = new TreeSet<>(s);
```


## Common Operations on TreeSet:

```java
import java.util.TreeSet;

public class TreeSetDemo {

    public static void main(String[] args) {

        // Creating TreeSet
        TreeSet<Integer> set = new TreeSet<>();

        // add()
        set.add(50);
        set.add(20);
        set.add(70);
        set.add(40);
        set.add(10);

        System.out.println("TreeSet: " + set);

        // contains()
        System.out.println("Contains 40? " + set.contains(40));

        // first() and last()
        System.out.println("First element: " + set.first());
        System.out.println("Last element: " + set.last());

        // higher() and lower()
        System.out.println("Higher than 40: " + set.higher(40));
        System.out.println("Lower than 40: " + set.lower(40));

        // remove()
        set.remove(20);
        System.out.println("After removing 20: " + set);

        // pollFirst() and pollLast()
        System.out.println("pollFirst(): " + set.pollFirst());
        System.out.println("pollLast(): " + set.pollLast());

        System.out.println("After polling: " + set);

        // iteration
        System.out.println("Iterating elements:");

        for (Integer num : set) {
            System.out.println(num);
        }
    }
}
// Output:
// TreeSet: [10, 20, 40, 50, 70]
// Contains 40? true
// First element: 10
// Last element: 70
// Higher than 40: 50
// Lower than 40: 20
// After removing 20: [10, 40, 50, 70]
// pollFirst(): 10
// pollLast(): 70
// After polling: [40, 50]

// Iterating elements:
// 40
// 50
```
